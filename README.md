# Poke-card

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```
### Para rodar o falso-backend instale 

```
npm install -g json-server
```
### E para rodar o servidor uitlize

```
npm start in server folder
```
